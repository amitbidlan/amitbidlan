### [Synaptic](https://github.com/amitbidlan/zistica-synaptic) — local-first AI agent observability

[![CI](https://github.com/amitbidlan/zistica-synaptic/actions/workflows/ci.yml/badge.svg)](https://github.com/amitbidlan/zistica-synaptic/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://github.com/amitbidlan/zistica-synaptic/blob/main/LICENSE)
[![Stars](https://img.shields.io/github/stars/amitbidlan/zistica-synaptic?style=social)](https://github.com/amitbidlan/zistica-synaptic/stargazers)

[![Synaptic dashboard preview](https://raw.githubusercontent.com/amitbidlan/zistica-synaptic/main/assets/dashboard-trace.png)](https://github.com/amitbidlan/zistica-synaptic)

</div>

> Add 2 lines of code, see every step your agent takes. Runs entirely on your laptop. No account, no data leaves your machine.

A CCTV camera for your AI agents — record every LLM call, tool invocation, and decision; replay them in a local dashboard; debug what went wrong before your users do.

- 🐍 Python + 📘 TypeScript SDKs with identical wire format
- ⚡ FastAPI ingest, DuckDB storage, single Docker image
- 🎨 Next.js 14 dashboard with span timeline
- 🦜 LangChain & 👥 CrewAI integrations (zero-config via `SYNAPTIC_TRACING=true`)
- ✅ 116 tests across packages, CI on every push
- 🔒 Local-first — your data never leaves your machine
